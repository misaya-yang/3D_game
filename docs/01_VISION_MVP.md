## 2. 核心游戏循环

### 2.1 分钟循环（战斗）

```
发现/锁定 → 连段/技能/闪避格挡
         → 命中: hitstop · 飘字 · SFX · 暴击震屏
         → 击杀 → 掉落/修为/任务 → HUD 反馈
```

### 2.2 小时循环（成长）

```
清怪攒修为与材料 → 回城炼丹/精炼/换装 → 层内升级或尝试突破
                 → 成功：属性跃迁 + 解锁区域/功法
                 → 失败：心魔 + Blockers 复盘 → 再准备
```

> **禁止**以打坐挂机作为主成长路径（打坐玩法延后；小时环以战斗/任务/丹药推进）。

### 2.3 会话循环（探索）

```
接主线/支线 → 地图节点 → 精英/秘境/奇遇 → 交付 · AutoSave → 下一目标钩子
```

### 2.4 再玩一把钩子

| 尺度 | 钩子 |
|------|------|
| 会话末 | 任务追踪 + 距突破 XP |
| 跨会话 | meta：境界、地图、`mainQuestIndex` |
| 未闭环 | 心魔剩余、黑风 `dungeonCheckpoint` |
| Build | 灵根被动差 → 第二周目 |

### 2.5 建议单次会话 45–75 min

| 分钟 | 活动 | 情绪 |
|------|------|------|
| 0–10 | 补给、任务、检查突破 | 计划 |
| 10–35 | 野外+采集+支线 | 心流 |
| 35–50 | 精英/秘境 | 紧张 |
| 50–65 | 突破 / BOSS | 高峰 |
| 65–75 | 交付、炼丹、存档 | 回落 |

## 5. MVP 垂直切片（Vertical Slice）

**25–40 分钟切片**（G-VS-*；**PR-8 合并前不做最终体验验收**）：

1. 新游戏 → 选灵根（三路线文案）→ 进入青石原  
2. **完成**移动/战斗教程（`TutorialManager` 写入完成态；DEV 可用 `/tutorial_skip`）  
3. 击杀 ≥5 练气妖兽；伤害字可见（hitstop 可选占位）  
4. 接主线任务，对话 NPC  
5. 使用回血丹、装备武器  
6. 学 1 个功法并释放  
7. 修为升级（GM 可加速）  
8. 存读档：背包/任务/教程完成态一致  

切片通过后，才允许横向铺内容。

**新手前 15 分钟必达反馈**：灵根文案 → 移动 → 第一击飘字 → 击杀修为 → 对话镜头 → 回血丹（满血不消耗）。

## 6. 难度与受众

- **默认难度**：中等  
- **操作**：键鼠为主；手柄 **战斗 + HUD + 背包 + 对话** 必做；炼丹/地图可键鼠优先  
- **死亡惩罚**：当前层修为 **5%**，回最近传送点并回满 HP/Mana；**不掉装备**  
- **突破失败**：扣 20% 当前层修为 + 30s 心魔（伤害 -10%）；**不掉境界**；**不消耗突破丹**（成功才扣）  
- **筑基主线 pity**：`quest_main_01_08` 首次合规突破强制成功（见 `05`）  
- **金丹突破**：无 pity，可失败重试

## 7. 性能预算（MVP）

| 指标 | 最低配置目标 | 推荐配置目标 |
|------|--------------|--------------|
| 分辨率/画质 | 1080p 中 | 1440p 高 |
| FPS | ≥30 | ≥60 |
| 场景加载 | <15s | <8s |
| 内存 | <6GB | <8GB |
| 同屏敌人 | ≤15 普通 或 1 BOSS | 同左 |

### 7.1 G07-03 边界与性能证据（2026-07-15）

统一边界目录 `MvpBoundaryCatalog` 固定 8 项；目标用例直接覆盖高频拒绝路径，其余复用
已签核的系统回归：

| 边界 | 结果 | 回归来源 |
|------|------|----------|
| 背包满 / 掉地 180s | 通过 | G07-03 满包拒绝 + G04-04 世界掉落 |
| 技能无蓝 / CD | 通过 | G07-03 无蓝不进状态并发 key + G03-02 CD |
| 装备境界不足 | 通过 | G07-03 拒绝并发 `ui_equipment_realm_required` |
| 突破中受击 | 通过 | G02-01 五拍期间无敌回归 |
| 任务重复接取 | 通过 | G07-03 第二次 `Accept` 返回 false |
| 对话切场景 | 通过 | G07-03 active scene 变化后强制结束并回 Playing |
| 敌人目标死亡 | 通过 | G04-01 Return 状态回归 |
| 存档盘满/损坏 | 通过 | EditMode SaveManager 损坏/写入失败安全回归 |

性能采样在 Apple M4、Unity 6000.5.3f1、Metal 图形设备、1920×1080 Medium、青石
切片场景进行：预热 30 帧后采样 120 帧，场景加载 **0.314s**，平均 **620.3fps**，
最差帧 **1.76ms**，峰值已分配内存 **220.9MiB**，同屏敌人不超过 15。该值是
Editor batchmode 灰盒切片证据，不外推为最终发行包基准；但已满足本节最低 30fps、
15s、6GB 门槛。

稳定性采用确定性加速巡检：推进昼夜、天气与技能冷却 3,600 个模拟秒，期间执行
60 次保存、6 次加载、60 次物品增删和重复任务拒绝，结果 **0 Error / 0 Exception**，
峰值内存 **228.5MiB**。这不是 60 分钟真人墙钟录像；最终新档真人走查仍归
G07-04，总验收不得把本代理记录冒充真人签核。
